## Idea keymap
映射到 Idea 默认快捷键

 `Caps + R` extend selection
 
 `Caps + N` go to file
 
 `Caps + G` navigate to declaration
 
 `Caps + F` back
 
 `Caps + Q + J` move caret to previous word
 
 `Caps + Q + L` move caret to next word
 
 `Caps + E` rename
 
 `Caps + C` comment line
 
 `Caps + V` reformat code
 
 `Caps + R` run
 
[Translation](https://plugins.jetbrains.com/plugin/8579-translation) 插件

 `Caps + T` translate
